# References

## Hardware and camera

1. NVIDIA Jetson AGX Orin product page. Jetson AGX Orin Developer Kit and AGX Orin 64GB list up to 275 TOPS and Ampere GPU specifications.
2. e con Systems AR0234 GMSL2 global shutter camera pages for Jetson AGX Orin. These describe AR0234 global shutter, GMSL2, Jetson support, external trigger, and multi camera synchronization variants.
3. Leopard Imaging LI AR0234CSC GMSL2 120H announcement, January 2026. This describes a 2MP global shutter GMSL2 8 camera system for Orin platforms with 1920 x 1200 RAW output, multi camera triggering, and up to 8 simultaneous cameras.

## NVIDIA software

1. NVIDIA VPI documentation, 2026. VPI is NVIDIA's vision programming interface for image processing and computer vision on NVIDIA embedded and discrete devices.
2. NVIDIA DeepStream SDK documentation. DeepStream provides multi sensor video, audio, image, and AI analytics pipelines based on GStreamer.
3. NVIDIA Holoscan Sensor Bridge documentation, 2026. Holoscan Sensor Bridge documents sensor processing examples including Jetson AGX Orin support.

## Reconstruction

1. COLMAP documentation. COLMAP is a general purpose Structure from Motion and Multi View Stereo pipeline with GUI and command line interface.
2. OpenMVS GitHub. OpenMVS targets the Multi View Stereo part of the photogrammetry chain.
3. OpenMVG documentation. OpenMVG provides command line tools for feature computation, matching, and SfM.
4. Nerfstudio documentation. Nerfstudio provides an end to end API for creating, training, and testing NeRFs and includes Gaussian Splatting workflows.
5. gsplat documentation. gsplat is a CUDA accelerated differentiable rasterization library for 3D Gaussian Splatting with Python bindings.
